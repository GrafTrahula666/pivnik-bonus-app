# Пивник | Бонусы

Первый разворачиваемый каркас Telegram Mini App программы лояльности бара «Пивник».

## Уже работает

- клиентский, барменский и административный экраны кликабельного MVP;
- динамический QR с обновлением каждые 30 секунд;
- статусы, прогресс, начисление и списание бонусов в демо-режиме;
- запуск внутри Telegram Mini App и обычного браузера;
- Node.js-сервер, готовый к Railway;
- `/api/health` для проверки сервера и PostgreSQL;
- подготовленные переменные окружения без опубликованных секретов.

## Локальный запуск

```bash
npm install
npm start
```

Затем открыть `http://localhost:3000`.

## Railway

1. Создать проект из этого GitHub-репозитория.
2. Railway автоматически определит Node.js и выполнит `npm start`.
3. После первого запуска открыть Settings → Networking → Generate Domain.
4. Полученный HTTPS-адрес будет использоваться в BotFather как URL Mini App.

## Безопасность

Никогда не добавлять токен Telegram в код или GitHub. Он указывается только в Railway Variables под именем `TELEGRAM_BOT_TOKEN`.
