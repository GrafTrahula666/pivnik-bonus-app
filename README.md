# pivnik-bonus-app

Telegram Mini App программы лояльности бара «Пивник».

## Что внутри

- клиентский экран с балансом, статусом, историей и QR;
- экран бармена для начисления и списания бонусов;
- админка с ролями, журналом и визуальным конструктором;
- PostgreSQL backend;
- загрузочный экран на основе фирменного ночного арта.

## Локальный запуск

1. Создать `.env` по примеру `.env.example`.
2. Указать `DATABASE_URL`, `TELEGRAM_BOT_TOKEN`, `OWNER_TELEGRAM_ID`, `SESSION_SECRET`.
3. Установить зависимости:

```bash
npm install
```

4. Запустить проект:

```bash
npm run dev
```

## Проверка синтаксиса

```bash
npm run check
```
