# Пивник — Telegram Mini App

Программа лояльности бара «Пивник» для клиентов, сотрудников и владельца.

## Реализовано

- авторизация через Telegram Mini App;
- клиентский бонусный баланс и статусы;
- постоянный персональный QR-код и короткий код;
- начисление и мгновенное списание бонусов;
- уведомления клиенту после операции;
- история начислений, списаний и корректировок;
- роли клиента, сотрудника, наблюдателя и владельца;
- отметка операций свыше 3000 ₽;
- отдельная корректировка баланса с обязательной причиной;
- перевыпуск персонального QR из админки;
- справка, FAQ, правила бета-теста и согласие при первом запуске;
- фирменный загрузочный экран;
- акция «15-й литр в подарок» с учётом объёма и выдачей по 0,5 или 1 л.

## Запуск

```bash
npm install
npm start
```

Обязательные переменные окружения:

- `DATABASE_URL`
- `TELEGRAM_BOT_TOKEN`
- `OWNER_TELEGRAM_ID`
- `SESSION_SECRET`

Для локального демонстрационного режима:

- `ALLOW_DEMO=true`

## Проверка

```bash
npm run check
```
