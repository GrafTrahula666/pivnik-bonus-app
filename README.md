# Пивник — исправленный пакет

В этой версии изображение загрузочного экрана встроено прямо в `index.html`.
Отдельная папка `assets` не требуется, поэтому изображение не потеряется при загрузке файлов через GitHub.

## Проверка

```bash
npm install
npm run check
npm start
```

Файлы `DOCS-*` содержат подготовленные правила, FAQ и технический план.
