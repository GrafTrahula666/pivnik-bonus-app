# Пивник — финальная закрытая бета

Telegram Mini App программы лояльности бара «Пивник».

## Готово

- бонусные статусы, начисление и мгновенное списание;
- постоянный QR и короткий код;
- 100 приветственных бонусов новым клиентам;
- 15-й литр в подарок;
- роли клиента, бармена, партнёра-наблюдателя и владельца;
- PIN сотрудников и управление текущей сменой;
- магазин за бонусы;
- ежемесячная таблица лидеров;
- отмены операций с причиной и лимитом 3 на сотрудника за смену;
- подозрительные операции свыше 3000 ₽;
- неизменяемая история и ручные корректировки только с причиной;
- премиальный чёрно-белый интерфейс;
- загрузочный экран со звёздным городом и анимациями.

## Запуск

```bash
npm install
npm start
```

Обязательные переменные Railway:

- `DATABASE_URL`
- `TELEGRAM_BOT_TOKEN`
- `OWNER_TELEGRAM_ID`
- `SESSION_SECRET`

Для локального демонстрационного режима:

- `ALLOW_DEMO=true`

## Обновление существующего проекта

Загрузите содержимое этой папки в корень GitHub-репозитория с заменой файлов. Railway выполнит автоматический деплой. База обновится сама при запуске.

## Проверка

```bash
npm run check
```

Сценарий проверки на двух телефонах находится в `TEST-CHECKLIST-TWO-PHONES.md`.


## V11: управление контентом

В панели владельца появился раздел «Акции и магазин». Владелец может создавать, изменять, скрывать, сортировать и удалять карточки, загружать JPG/PNG/WEBP или указывать HTTPS-ссылку, а также менять цену товаров в бонусах. Изображения уменьшаются в браузере и сохраняются в PostgreSQL. Стартовая цена крафта — 600 бонусов. Партнёрский обзор видит каталог, но не имеет кнопок редактирования.


## V11 — профиль, аватары и конфиденциальность

- После принятия правил приложение один раз предлагает настроить профиль.
- Выбор без случайной выдачи: мужской силуэт, женский силуэт, фото Telegram или один из 19 бесплатных аватаров.
- Возрастная группа указывается по желанию и используется только для внутренней статистики, в профиле не отображается.
- Настройки публичности: имя, аватар, сумма в таблице лидеров, статистика и возможность открыть подробный профиль.
- Настройки можно повторно открыть нажатием на аватар или кнопку «Аватар и конфиденциальность».
- Аватары отображаются в профиле, таблице лидеров, текущей смене и карточке найденного клиента.


## V12

- Нижняя навигация закреплена над системной областью и больше не уезжает при прокрутке админки.
- Владелец получает постоянный баланс 9 999 999 999 999 бонусов, усиленную анимированную космическую рамку и уникальное достижение «Создатель» приложения.
- Пользователь с ролью «Партнёрский обзор» получает постоянный баланс 9 999 999 999 999 бонусов и огненную анимированную рамку.
- Безлимитные профили не теряют бонусы при списании и покупке товаров.


## V12.4 — исправление аватаров и литров

- Все изображения аватаров включены в архив обновления.
- При запуске приложение восстанавливает литры из завершённых операций, если отдельная таблица прогресса оказалась пустой.
- В каждой новой операции сотрудник обязан выбрать объём разливного или явно нажать «Без разливного».
- Это предотвращает случайные операции без учёта литров.

## V13 — магазин, обращения и админ-панель

- Сотрудник может открыть клиентскую часть приложения независимо от смены и состояния PIN. Рабочая авторизация проверяется только в разделе бармена.
- На главной показывается витрина физических товаров с интервалом 10 секунд; оформления профиля в неё не попадают.
- Магазин разделён на категории, у каждой позиции есть изображение или автоматически созданная обложка.
- Клиент может написать владельцу по конкретному товару или оставить обращение внутри приложения.
- На главном экране админ-панели показываются только 5 последних пользователей, транзакций и обращений; полные списки открываются отдельно.
- Анне Берман один раз устанавливается 1 000 000 бонусов для бета-теста и персональная анимированная рамка.
- QR-карту можно сохранить как изображение. Для появления нативных кнопок Wallet настройте `APPLE_WALLET_ISSUER_URL` и `GOOGLE_WALLET_ISSUER_URL`.

Дополнительные переменные V13:

- `OWNER_TELEGRAM_USERNAME` — username владельца без `@` для кнопки личного сообщения;
- `ANNA_TELEGRAM_ID` — точный Telegram ID Анны Берман, чтобы правило миллиона применилось без зависимости от имени профиля;
- `APPLE_WALLET_ISSUER_URL` — URL внешнего сервиса выпуска Apple Wallet pass;
- `GOOGLE_WALLET_ISSUER_URL` — URL внешнего сервиса выпуска Google Wallet pass.

## V14 — Android-safe загрузка и достижения

V14 открывает клиентский профиль до необязательных запросов, добавляет восстановление после ошибок Android WebView и предварительный раздел достижений. Первые 30 зарегистрированных пользователей получают легендарное достижение «Тестировщик» и 150 бонусов один раз. Миграции выполняются автоматически при старте; дополнительные переменные окружения не требуются.
