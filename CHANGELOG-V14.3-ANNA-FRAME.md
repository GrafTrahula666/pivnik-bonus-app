# V14.3 — рамка Анны

- Персональная рамка Анны восстанавливается по Telegram ID, username, сохранённой рамке или уникальной записи `anna-senior-beta-million`.
- Если точные идентификаторы не заданы, допускается безопасный резервный поиск единственной сотрудницы с именем Анна/Аня.
- Смена роли через админ-панель больше не сбрасывает рамку Анны в `none`.
- После деплоя миграция автоматически устанавливает `profile_frame = anna` уже найденной Анне.
- Добавлена необязательная переменная `ANNA_TELEGRAM_USERNAME`.
- Версии клиентских файлов повышены до 14.3 для сброса кэша Android WebView.
