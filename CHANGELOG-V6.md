# Пивник — обновление v6

## Готово

- постоянный персональный QR-код клиента;
- короткий код формата `PVK-XXXX-XXXX` для ручного ввода;
- перевыпуск QR владельцем, старый код после перевыпуска не работает;
- списание бонусов без подтверждения клиента;
- начисление и списание фиксируются в неизменяемой истории;
- операции свыше 3000 ₽ помечаются как подозрительные;
- уведомление владельцу о подозрительной операции;
- ручная корректировка баланса только с причиной;
- экран принятия правил при первом запуске;
- обновлённые справка, FAQ, правила и конфиденциальность;
- визуальная полировка клиента, бармена, админки, истории и модальных окон;
- загрузочный экран со звёздами, гличем вывески, движением человека и дымом;
- редкие падающие звёзды без визуальной перегрузки;
- улучшенные состояния онлайн/офлайн, ошибок и успешной операции;
- защита от повторного нажатия при проведении операции;
- миграции существующей базы выполняются автоматически при запуске.

## Не требует ручной миграции базы

Новые поля и индексы создаются через `ALTER TABLE ... IF NOT EXISTS`. Существующие пользователи автоматически получают постоянные QR-коды при первом запуске новой версии.
