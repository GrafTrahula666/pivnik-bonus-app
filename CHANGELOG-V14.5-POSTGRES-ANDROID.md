# V14.5 — PostgreSQL / Android hotfix

## Исправлена фактическая причина зависания

Ошибка `inconsistent types deduced for parameter $2` возникала при первой регистрации:
один параметр SQL одновременно использовался для поля `bonus_earned` типа INTEGER
и поля `balance_after` типа BIGINT.

Теперь для этих значений используются разные параметры с явными типами.

## Дополнительная защита входа

- Выдача бета-достижения, правила Анны и создание персонального QR выполняются в отдельных SAVEPOINT.
- Ошибка необязательной награды больше не блокирует авторизацию и клиентский профиль.
- При временной серверной ошибке сохранённая сессия не удаляется.
- Одновременный повторный запуск загрузки заблокирован, чтобы Android WebView не отправлял несколько параллельных запросов.
- Сырые ошибки PostgreSQL больше не показываются клиенту; вместо них выводится безопасное сообщение и код ошибки для журнала Railway.
- Версия клиентских файлов поднята до 14.5 для сброса кэша Telegram WebView.
