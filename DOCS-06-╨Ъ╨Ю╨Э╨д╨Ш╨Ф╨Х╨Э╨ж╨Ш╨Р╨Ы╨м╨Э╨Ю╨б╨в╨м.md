# Конфиденциальность и условия бета-теста

## Какие данные используются

Приложение может использовать:

- Telegram ID;
- имя и username из Telegram;
- технические данные авторизации Telegram Mini App;
- бонусный баланс;
- историю операций;
- роль пользователя в системе.

## Для чего данные используются

Данные применяются для:

- авторизации пользователя;
- отображения профиля;
- начисления и списания бонусов;
- ведения журнала операций;
- контроля доступа сотрудников и администраторов.

## Хранение и доступ

- данные хранятся в базе приложения;
- доступ к административным функциям ограничен ролями;
- изменения баланса фиксируются в журнале;
- служебные действия сотрудников могут логироваться.

## Ограничение ответственности в бета-тесте

Бета-версия предоставляется в тестовом режиме. Возможны:

- ошибки работы интерфейса;
- временная потеря части функциональности;
- задержки обновления информации;
- необходимость повторной операции;
- изменение функций и правил по мере развития приложения.

Используя приложение во время бета-теста, пользователь соглашается с тем, что сервис находится в режиме испытаний.

## Практический смысл для клиента

- если что-то отображается неверно, нужно сообщить владельцу или сотруднику;
- итог по спорным случаям определяется по журналу операций и серверным данным;
- передавать свой QR-код другим людям не рекомендуется.
