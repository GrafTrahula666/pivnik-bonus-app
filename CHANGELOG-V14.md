# V14 — Android-safe загрузка и предварительная система достижений

## Android и загрузка

- Внешний Telegram SDK больше не блокирует отрисовку первого экрана.
- Тяжёлые изображения загрузочного экрана вынесены из HTML в отдельные локальные файлы.
- Основной HTML уменьшен примерно с 300 КБ до 52 КБ.
- Добавлены таймауты запросов, повтор безопасных GET-запросов и ожидание Telegram initData на Android.
- Профиль открывается до загрузки необязательных блоков: магазина, рейтинга, акций, смены и Wallet.
- Ошибка одного необязательного блока больше не оставляет всё приложение на вечной загрузке.
- На экране ошибки доступны кнопки «Повторить» и «Открыть облегчённо».
- Для Android WebView отключены тяжёлые blur, backdrop-filter, mix-blend и непрерывные анимации.
- Старая или просроченная сессия сотрудника очищается и не блокирует клиентскую часть.
- HTML, CSS и JavaScript отдаются без устаревшего кэша; ресурсы имеют версию V14.

## Достижения

- Добавлен отдельный раздел «Достижения».
- Вкладки: обычные, редкие, эпические и легендарные.
- Карточки выводятся по три в ряд.
- Будущие достижения показаны размытыми карточками «В разработке».
- В профиле появилась витрина уже полученных медалей с разным размером по редкости.
- Первые 30 пользователей приложения получают легендарное достижение «Тестировщик».
- За «Тестировщика» один раз начисляется 150 бонусов, операция фиксируется в журнале.
- Для ещё не просмотренного достижения показывается полноэкранное поздравление.
- Уникальное достижение владельца «Создатель» получило знак треугольника с глазом.
- Добавлена серверная база для дальнейшего расширения каталога достижений без переделки профилей.
