# V12.2 — аватары и литровая шкала

- В архив возвращена папка `assets/avatars` со всеми 21 изображениями.
- Для аватаров добавлено обновление кеша и заметный резервный вариант.
- При запуске сервер сверяет литры с завершёнными операциями и восстанавливает значения, если таблица лояльности оказалась пустой.

- Сервер не принимает операцию без подтверждённого поля объёма.
